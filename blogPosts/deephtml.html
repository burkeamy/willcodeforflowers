HTML, the deep cuts

Nov. 7, 2020

Most people know about the h1 tag and a p tag. Even the nav tag and article, section, and aside. There are about 250 HTML5 tags and most people only use maybe 20. That means there are over 200 more available.

If you are reading this and you have no idea what I am talking about, this paragraph is for you. HTML or Hyper Text Markup Language is the main building block of the web. The latest version is 5. The short explanation is that HTML is made of tags nested in the greater than or less than sign. You can view the HTML of many websites by right clicking on the page and clicking on inspect. If you are looking for a more in depth description, check out w3schools.com.

Some of them, like the canvas tag, require javascript to work properly. There are a few API available ones like web storage and geolocation. These also work along with javascript.

I have been using the figure tag with an image tag and a figcaption tag. This allows you to group your image with a caption and make it accessible at the same time.

If you are writing a scientific or math related post, you may want to use sub sub and sup sup to help with writing numbers. You might want to include some underlined text with a u tag.

If you include the time tag in your date, users can add it to their calendar. I used this at the top of the page with the date.

There is also a code tag you can use to represent a section of code. Who knew?

Are you adding a quote to your page? You can use the q tag for a short quote or the blockquote tag for a long quote. Below, I used cite for the author. This will put it in italics. Just defining something? Use the dfn tag. (I used this above when I defined HTML)
